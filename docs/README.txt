Place your team contract and design documents in this directory.

The files in this directory should be:
- team-contract.pdf
- design-milestone.pdf
- design-revised.pdf

If you add any other documents to this directory, please add the filenames to
the above list.

To run the server: Run the main method in the CollaboardServer.java file.
To run the client: 
1. Change the "host" argument in the main method of the Client.java file to match the IP of the machine running the server. 
2. Run the main method. Each time it is run, a new Client socket connection is created.
